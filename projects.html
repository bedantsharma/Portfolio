Generative Adversarial Network

Generative Adversarial Networks (GANs) are a type of deep learning framework consisting of two neural networks: a generator and a discriminator. The generator learns to generate new data instances that mimic a training set of real data, aiming to fool the discriminator. Simultaneously, the discriminator learns to distinguish between real data from the training set and fake data generated by the generator.
